# うさマネキン

## Materials
- Hook　Size
  - #6 (3.5mm)
- Yarn
  - アクリル毛糸　並太　(15目/10cm)
  - 403 アイボリー
  - ダイソー

## Pattern
- 頭からそのまま胴体を編む。
- 手足はそれぞれ作って両端をボタンで留めて組み合わせる。
- 頭部のみ、立ち上がりせずにぐるぐる編む。
- 全て細編み。それ以外の箇所はメモ。
- 増し目減らし目は適時分散させる。
- メモ例
  - +: 細編み
  - v: 細編み2目の増し目
  - Λ: 細編み2目一度
  - +11v: 細編み14目、細編み2目の増し目
  - +12Λ: 細編み12目、細編み2目一度
